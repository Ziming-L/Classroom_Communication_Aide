# Classroom_Communication_Aide
This is a project for Technology for Social Good class

1. In the folder with our source code.
2. Open 2 terminals
3. Cd to backend and frontend, and type `npm install` to install both dependencies
4. Create a file called ".env" on backend folder and paste the following:
PORT=5100
5. on backend terminal type: node server.js
6. on frontend terminal type: npm run dev
